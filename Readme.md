# Random Topics I Learn Deeply

This repository is my learning space for development internals, system design, AI, and other topics that help me grow from developer toward architect.

The focus is not just "what is this thing?" The focus is:

```text
How does it work internally?
Why was it designed this way?
What trade-offs does it create?
How does it affect real systems?
How would I explain it clearly in a design review?
```
